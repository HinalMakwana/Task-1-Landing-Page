Landing Page Design

A visually appealing **landing page** inspired by Netflix's style.  
This project demonstrates responsive front-end design using **HTML5** and **CSS3**, featuring a countdown timer layout and a styled login form.

Features

- Modern and clean landing page design  
- Split layout with countdown section and login form  
- Gradient overlay background with custom imagery  
- Simple, minimal login form with hover and focus styles  
- Fully responsive layout using Flexbox
